# Send His Word Ministries Website

Full Vite + React website package for launch.

## Preview

Open `dist/index.html` for an immediate static preview.

## Development

```bash
npm install
npm run dev
```

## Deployment

Vercel settings:

```txt
Framework: Vite
Build Command: npm run build
Output Directory: dist
```
